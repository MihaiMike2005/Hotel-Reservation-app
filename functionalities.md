Hotel Booking Application

This application allows managing hotel room bookings, as well as adding additional services (e.g. spa, gym, airport pickup). The application will be developed in C++ using object-oriented programming principles.

1. Room Management: • Storing information about rooms (room number, room type, number of beds, price per night). • Displaying all available rooms. • Checking the availability of a room for a specific time period.

2. Customer Management: • Entering customer details (first name, last name, phone number). • Searching for a customer by name or phone number. • Storing and displaying the booking history of a customer.

3. Making a Reservation: • Selecting the desired room. • Automatically calculating the total cost based on the number of nights. • Issuing the booking confirmation.

4. Additional Services: • Listing available services (e.g., pool access, breakfast, Wi-Fi, spa). • Adding one or more services to a reservation. • Calculating the additional cost generated by services.

5. Billing: • Generating the total invoice (room + services). • Displaying the invoice in text format. • Option to save the invoice to a local file.

6. Data Persistence: • Saving room and booking data to a file. • Automatically loading data when the application starts.

7. User Interface: • Interactive console menu. • Easy navigation through options. • Clear confirmation or error messages.
